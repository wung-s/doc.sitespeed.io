Examples

Here's a couple of examples of what the result looks like when you analyze a page.

Testing one site, done exactly this way (with User-Agent as an Iphone, fetching timing metrics using Chrome & take screenshots):

./bin/sitespeed.io -u http://www.cybercom.com  -c chrome -a iphone  -k true

Testing multiple sites, the top ranked sites from Alexa (using Chrome for timings, maximum 10 pages for each site):

./bin/sitespeed.io-sites -j 10 -i test/alexa-top10.txt -c chrome -z 1
