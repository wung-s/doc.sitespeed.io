Analyze your website speed and performance

Sitespeed.io is an open source tool that helps you analyze your website speed and performance based on performance best practices and metrics. It collects data from multiple pages on your website, analyze the pages using the rules and output the result as HTML or JUnit XML.

You can analyze one site, analyze & compare multiple sites or let it run in you CI tool to make sure that your site is always built the best way for speed.

Download or clone or fork the project at Github.

The 2.0 release

The 2.0 release is here. Almost everything has been reworked since 1.8 so please upgrade to the latest release

You can now configure the output, collect timing metrics from the Navigation Timing API & test your site for mobile. You can read about all the new things here.

Psst, the next version of sitespeed.io will support a Jenkins plugin, making it even easier than today to keep track of our site.

Development

Checkout issues and coming features at Github and follow sitespeed.io on Twitter to get information about new releases.

Sitspeed.io is developed & tested on Mac OS X 10.8.5 & Ubuntu Linux 12.04 and should run on Windows if you use Git Bash.

Why sitespeed.io?

  1. Check your site against the latest web performance best practice rules.
  2. Analyze many pages on your site, not just one!
  3. Test for desktop or mobile!
  4. Fetch Navigation Timing API metrics and your own custom measurements.
  5. Compare sites, keep track of your competitors.
